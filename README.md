# LevelConverterForTrackballModule

## 概要
トラックボールモジュール用のレベル変換基板です。

2.7Vのレギュレーターと双方向レベル変換ICを搭載しており、トラックボールモジュールに電源供給するとともにPro Microなどのコントローラーにあわせて信号をレベル変換します。
